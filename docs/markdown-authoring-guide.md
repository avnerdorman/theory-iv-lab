# Markdown Authoring Guide for Music Theory IV

## 1. What Goes in Markdown

- All lesson prose  
- Headings (#, ##, ###)  
- Lists  
- Block quotes  
- Footnotes  
- Diagrams (ASCII or embedded SVG)  
- Links  
- Simple inline <span> elements

## 2. What Should Be HTML

- `<div class="lesson-content">` wrappers  
- `<iframe>` for labs  
- Special formatting (flexboxes, alignment)  
- Figure blocks
